# Role-based inactivity timeout + env spremenljivke

## Spremembe

### app/config.py
- `INACTIVITY_TIMEOUT_MINUTES` — privzeto **30** (teacher)
- `INACTIVITY_TIMEOUT_ADMIN_MINUTES` — privzeto **20** (admin/vodstvo)

### app/main.py
- Na `/` poti se pogleda user iz baze prek `user_id` piškotka
- Če je vloga `admin` ali `vodstvo` → uporabi `INACTIVITY_TIMEOUT_ADMIN_MINUTES`
- Sicer → uporabi `INACTIVITY_TIMEOUT_MINUTES`
- Vrednost se pretvori v ms in poda v template

### .env
```
INACTIVITY_TIMEOUT_MINUTES=30
INACTIVITY_TIMEOUT_ADMIN_MINUTES=20
```

## Delovanje
- **Teacher (navaden user)**: 30 min neaktivnosti → odjava
- **Admin / Vodstvo**: 20 min neaktivnosti → odjava
- Obe vrednosti nastavljivi v `.env`
