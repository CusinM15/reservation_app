# Odgovor — prostori fix + DB password fix

## Prostori — najden vzrok
`"".split(",")` v Pythonu vrne `[""]` (ne `[]`). Fix v config.py: filtriranje praznih vrednosti.

## DB password — deployment crash
- sola-config.yaml in .env imata dobesedno `***` namesto pravega gesla
- `kubectl apply -f sola-config.yaml` je prepisal ConfigMap z `***`
- Novi podi crashtajo: password authentication failed
- Stari podi še delajo (imajo staro pravo geslo v env)

Popravi ročno:
```
PASS=$(kubectl get secret sola-postgresql -n sola -o jsonpath='{.data.password}' | base64 -d)
sed -i "s|sola:.*@sola-postgresql|sola:${PASS}@sola-postgresql|g" sola-config.yaml .env
kubectl apply -f sola-config.yaml
kubectl rollout restart deployment/sola-app -n sola-app
```
