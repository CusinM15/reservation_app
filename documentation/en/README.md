# School App – Reservation & Assessment Management

A web application for OŠ Toneta Čufarja Jesenice for room reservations (tablets, computer room, boat, and home economics classroom) and assessment scheduling.

---

## ⚠️ Problem: Windows 10 is no longer secure, Windows 11 won't run on old computers

The school has several working computers that are too old for Windows 11 (TPM 2.0, CPU requirements), while Windows 10 **has not received security updates since October 2025**. Leaving them on Windows 10 is a security risk.

**Options:**

| Option | Description |
|--------|-------------|
| **Linux Mint** | The most "Windows-like" — if you can use Windows, you'll feel right at home. Lightweight, works on old hardware. |
| **Zorin OS** | Even more Windows-like (same layout). "Zorin Connect" links with your phone. Also lightweight. |
| **Local server** | Turn one old computer into a server — install Linux, run the app, and share it across the school. |

---

## 🖥️ Local server — the harder path

If you turn an old computer into a server, you get full control — the app runs on-site, data is local, no monthly costs. However, it requires knowledge of:

- Linux installation
- Docker and docker-compose
- Network configuration (DHCP reservation, open ports)
- Maintenance (updates, backups, troubleshooting)
- Restart after power outages

**Detailed instructions** are available in the [documentation](main.md).

---

## ☁️ Easier option — cloud hosting

Since a local server requires quite a bit of knowledge and ongoing maintenance, there's a simpler solution: **put the app in the cloud**. The provider handles everything — server, database, backups, security. Pay ~€7–15 per month with zero maintenance work.

More details in [migracija_oblak.md](migracija_oblak.md) — a document comparing all options with exact pricing and pros/cons.
